General & Installation

The Ace Text Highlighting plug-in extends Docmenta's internal text editor by syntax highlighting functionality. The syntax highlighting is based on the Open Source code editor Ace (see the Ace homepage).
The Ace Text Highlighting plug-in can be downloaded from http://www.docmenta.org. For instructions on installing the plug-in, see 'Installing Plug-ins' [Docmenta User Manual]. After the plug-in has been installed, the user-specific configuration of the plug-in can be done in the user profile (see Configuration).
